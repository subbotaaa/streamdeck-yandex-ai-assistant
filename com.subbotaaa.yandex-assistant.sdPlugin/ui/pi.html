API-ключ сервисного аккаунта Yandex Cloud с ролями ai.speechkit-stt.user, ai.speechkit-tts.user, ai.languageModels.user

Идентификатор каталога в Yandex Cloud (консоль → каталог → ID)

  1. Войдите в консоль Yandex Cloud (нужен аккаунт Яндекса и активный платёжный аккаунт — создаётся там же, в разделе «Биллинг»).
  2. Откройте или создайте каталог (folder). Его идентификатор вида b1g... показан в списке каталогов и в адресной строке — скопируйте его в поле Folder ID.
  3. Слева выберите Сервисные аккаунты → «Создать сервисный аккаунт». При создании добавьте роли: ai.speechkit-stt.user, ai.speechkit-tts.user, ai.languageModels.user.
  4. Откройте созданный аккаунт → вкладка «API-ключи» → «Создать API-ключ». Скопируйте секрет (строка вида AQVN...) — он показывается только один раз! Вставьте его в поле API-ключ.
  5. Нажмите «Проверить подключение» внизу этой панели.

Все модели работают с вашим API-ключом Yandex Cloud и оплачиваются по тарифам Foundation Models / AI Studio.

Имя модели из консоли Yandex Cloud (раздел AI Studio → Модели) или полный URI. К короткому имени автоматически добавится /latest.

Если пишете свой промпт, добавьте в него: «У тебя нет доступа к интернету, отвечай из собственных знаний» — иначе модель любит отказываться от ответа.
