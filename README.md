**Голосовой ИИ-ассистент на кнопке Stream Deck.** Зажмите кнопку, задайте вопрос голосом —
и получите ответ нейросети **голосом** или **текстом**. Работает на Yandex Cloud:
распознавание и синтез речи — SpeechKit, ответы — YandexGPT, Алиса AI, Qwen3, DeepSeek
и другие модели AI Studio на выбор.

> **English:** A push-to-talk voice AI assistant plugin for Elgato Stream Deck, powered by
> Yandex Cloud (SpeechKit STT/TTS + YandexGPT / Alice AI / Qwen / DeepSeek via Foundation
> Models). Windows only. UI is in Russian.

## Возможности

- 🎙️ **Push-to-talk**: зажали кнопку → говорите → отпустили → ответ. Или режим
  «нажать — старт, нажать — стоп».
- 🔊 **Ответ голосом** (14+ голосов SpeechKit, регулируемая скорость) и/или
  💬 **текстом** — всплывающей карточкой с настраиваемым временем показа.
- 🧠 **Память диалога**: ассистент помнит последние реплики (настраивается, 0–20),
  можно задавать уточняющие вопросы.
- 🤖 **Выбор модели**: YandexGPT 5 Lite / Pro / 5.1, Алиса AI (+Flash), Qwen3 235B,
  DeepSeek V3.2 / V4 Flash, GPT-OSS 120B или любая своя по URI.
- 📓 **Журнал диалогов** — все вопросы и ответы в markdown-файле в «Документах».
- 🎛️ Выбор **микрофона и динамика**, звуковые сигналы записи, копирование ответа
  в буфер обмена.
- 🚫 **Без внешних зависимостей**: запись и воспроизведение — через встроенные
  средства Windows (WinMM), ffmpeg/sox не нужны.

## Как это работает

```
[зажали кнопку] → запись с микрофона (WAV 16 кГц, WinMM MCI)
[отпустили]     → SpeechKit STT (распознавание речи)
                → выбранная LLM (OpenAI-совместимый API Foundation Models)
                → SpeechKit TTS (озвучка) и/или карточка с текстом
```

## Требования

- Windows 10/11
- [Stream Deck](https://www.elgato.com/downloads) 6.9+ (проверено на 7.1)
- Аккаунт [Yandex Cloud](https://console.yandex.cloud/) с активным платёжным аккаунтом

## Установка

**Вариант 1 — готовый пакет.** Скачайте `com.subbotaaa.yandex-assistant.streamDeckPlugin`
из [релизов](https://github.com/subbotaaa/streamdeck-yandex-ai-assistant/releases)
и откройте двойным кликом — Stream Deck установит плагин сам.

**Вариант 2 — из исходников:**

```powershell
git clone https://github.com/subbotaaa/streamdeck-yandex-ai-assistant.git
cd streamdeck-yandex-ai-assistant
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

Скрипт поставит зависимости (нужен Node.js + npm), скопирует плагин в папку
Stream Deck и перезапустит приложение.

## Настройка Yandex Cloud (один раз, ~5 минут)

1. Войдите в [консоль Yandex Cloud](https://console.yandex.cloud/). Если аккаунта нет —
   зарегистрируйтесь и создайте платёжный аккаунт (раздел «Биллинг»).
2. Откройте или создайте **каталог** (folder). Скопируйте его **ID** (строка вида
   `b1g...` в списке каталогов) — это поле **Folder ID** в настройках кнопки.
3. Создайте **сервисный аккаунт**: IAM → Сервисные аккаунты → «Создать». Назначьте роли:
   - `ai.speechkit-stt.user` — распознавание речи;
   - `ai.speechkit-tts.user` — синтез речи;
   - `ai.languageModels.user` — языковые модели.
4. Откройте сервисный аккаунт → вкладка **«API-ключи»** → «Создать API-ключ».
   Скопируйте **секрет** (строка `AQVN...`, показывается один раз!) — это поле
   **API-ключ** в настройках кнопки.

Эта же инструкция встроена в настройки кнопки — кнопка «📖 Как получить API-ключ».

## Настройка кнопки

Перетащите действие **«Голосовой ассистент»** (категория «Yandex AI») на кнопку
и заполните настройки:

| Настройка | Описание | По умолчанию |
|---|---|---|
| API-ключ, Folder ID | из Yandex Cloud (см. выше) | — |
| Модель | YandexGPT / Алиса AI / Qwen3 / DeepSeek / GPT-OSS / своя | YandexGPT 5 Lite |
| Системный промпт | роль и стиль ответов | встроенный |
| Язык речи | русский / английский / автоопределение | русский |
| Микрофон / Динамик | конкретные устройства | системные |
| Режим записи | удерживать (push-to-talk) / нажать-старт-нажать-стоп | удерживать |
| Звуковой сигнал | бип при старте/окончании записи | вкл |
| Формат ответа | голосом / текстом / оба | голосом |
| Голос и скорость | 8 голосов SpeechKit, 0.5–2× | Алёна, 1.0× |
| Показ уведомления | 3–30 секунд или до клика | 5 секунд |
| Журнал диалогов | запись в `YandexAssistant-журнал.md` («Документы») | вкл |
| Память диалога | сколько пар «вопрос-ответ» помнить (0–20) | 5 |

После заполнения нажмите **«Проверить подключение»** — плагин сделает тестовый
запрос к выбранной модели.

## Использование

- **Удерживайте** кнопку и говорите (до 29 секунд — лимит синхронного распознавания
  SpeechKit), затем отпустите.
- Состояния кнопки: 🎙 фиолетовая — готов; 🔴 красная точка — идёт запись;
  ⚪ точки — обработка; 🟢 динамик — воспроизведение ответа.
- Нажатие во время воспроизведения обрывает звук и начинает новый вопрос.
- Уточняющие вопросы работают благодаря памяти диалога: «а расскажи подробнее»,
  «переведи это на английский» и т.п.

## Сколько это стоит

Оплачивается только использование API Yandex Cloud (тарифы AI Studio, с НДС,
могут меняться — [актуальный прайс](https://yandex.cloud/ru/docs/ai-studio/pricing)).
Ориентир: один голосовой вопрос-ответ с YandexGPT Lite ≈ 0,8 ₽, с Qwen3 235B ≈ 1,2 ₽
(распознавание ≈ 0,16 ₽ + модель + озвучка ≈ 0,4 ₽).

## Безопасность

⚠️ Stream Deck хранит настройки кнопок (включая API-ключ) в **открытом виде** в профиле
на диске. Используйте отдельный сервисный аккаунт только с тремя ролями выше и не
используйте этот ключ где-либо ещё. Плагин отправляет запросы только на
`*.api.cloud.yandex.net` / `llm.api.cloud.yandex.net`.

## Диагностика

- Логи: `%APPDATA%\Elgato\StreamDeck\logs\com.subbotaaa.yandex-assistant*.log`
- Ошибки API показываются всплывающей карточкой с текстом ошибки.
- «Речь не распознана» — говорите ближе к микрофону, проверьте выбранный микрофон
  в настройках и уровень в Параметрах Windows → Звук.
- Плагин не появился после установки — перезапустите Stream Deck.

## Структура проекта

```
com.subbotaaa.yandex-assistant.sdPlugin/
├── manifest.json          # манифест плагина (SDK v2, Node.js 20)
├── bin/plugin.js          # ядро: WebSocket-протокол Stream Deck + API Yandex Cloud
├── helpers/recorder.ps1   # запись с микрофона (WinMM MCI), персистентный процесс
├── helpers/play.ps1       # воспроизведение WAV с выбором устройства
├── helpers/devices.ps1    # перечисление аудиоустройств
├── helpers/toast.ps1      # всплывающая карточка уведомления (WinForms)
├── ui/pi.html             # Property Inspector (настройки кнопки)
└── imgs/                  # иконки
tools/gen-icons.mjs        # генератор иконок (чистый Node, без зависимостей)
install.ps1                # установка из исходников
```

## Разработка

```powershell
# иконки
node tools/gen-icons.mjs
# проверка синтаксиса
node --check com.subbotaaa.yandex-assistant.sdPlugin/bin/plugin.js
# установка локально
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

Технические заметки: плагин общается со Stream Deck по WebSocket-протоколу напрямую
(зависимость — только `ws`); запись звука — MCI-команды `winmm.dll` из персистентного
PowerShell-процесса; модели вызываются через OpenAI-совместимый эндпоинт
`llm.api.cloud.yandex.net/v1/chat/completions` (работает для всех моделей AI Studio,
включая открытые).
